Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ вниз от соседей равным 2em, при помощи свойства margin-boottom

Статья 2

Размер блока:
занимает всю доступную ширину
должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em, при помощи свойства margin-boottom. Картинка вставленна при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Заменяет всю доступную ширину. Высота определяется помещенным внутрь контентом
